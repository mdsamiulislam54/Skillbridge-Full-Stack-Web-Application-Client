# SkillBridge 
**Connect with Expert Tutors, Learn Anything**

SkillBridge is a full-stack tutoring platform that connects students with expert tutors. Students can browse tutors, book sessions, and leave reviews, while tutors can manage profiles and availability. Admins oversee the entire platform.

---

#  Live Demo

 Client: https://skillbridge-chi-seven.vercel.app

Server: https://skillbridge-app-api-inky.vercel.app 

---

#  Project Overview

SkillBridge provides a seamless experience for students and tutors:

- Students can find tutors easily
- Tutors can manage teaching professionally
- Admin can control and monitor the system

Built with modern full-stack technologies using scalable architecture.

---

#  Features

##  Public Features

- Browse tutors
- Search tutors by subject, price, rating
- Filter tutors by category
- View tutor profile
- View reviews
- Landing page with featured tutors

---

##  Student Features

- Register and login
- Book tutoring sessions
- View upcoming bookings
- View booking history
- Leave reviews
- Manage profile

---

##  Tutor Features

- Register and login
- Create tutor profile
- Update tutor profile
- Add availability slots
- View teaching sessions
- View ratings and reviews

---

##  Admin Features

- View all users
- Ban / Unban users
- Manage categories
- View all bookings
- Platform management
